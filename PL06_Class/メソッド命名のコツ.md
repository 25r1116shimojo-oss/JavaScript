# メソッド命名のコツ

センサーの判定を行うメソッドの命名には、**そのメソッドが何をするのか、そしてどのような結果を返すのか**を明確に示すことが重要です。
一般的な命名規則として、メソッドの動作と戻り値の型に合わせて、以下のようなパターンがよく使われます。

## 💡 判定メソッドの命名パターン

### 1. 戻り値が真偽値（`bool`）の場合

メソッドが「〜の状態であるか」「〜の条件を満たしているか」を判定し、`true` または `false` を返す場合は、以下のプレフィックス（接頭辞）を使います。

| プレフィックス | 意味 | 命名例 |
| :--- | :--- | :--- |
| **`is`** | **現在の状態**が〜であるか。 | `isDetected()` (センサーが検知しているか) |
| **`can`** | **動作が可能**であるか。 | `canStartMeasurement()` (測定を開始できる状態か) |
| **`has`** | **結果や要素**を保持しているか。 | `hasError()` (エラー状態を保持しているか) |
| **`should`** | **次の動作**を行うべきか。 | `shouldTriggerAlarm()` (アラームを鳴らすべきか) |
| **`check`** | **特定の条件**を確認する。 | `checkRangeLimit()` (範囲制限を超えていないか確認) |

### 2. 戻り値が測定値やステータスの場合

単に判定するだけでなく、特定の状態や数値を**取得**したり、**実行**したりする場合に使われます。

| プレフィックス | 意味 | 命名例 |
| :--- | :--- | :--- |
| **`get`** / **`retrieve`** | 測定値やステータスを**取得**する。 | `getSensorValue()` (センサーの現在の値を取得) |
| **`read`** | センサーから直接**読み込む**。 | `readTemperature()` (温度センサーから値を読み込む) |
| **`calculate`** | 判定に必要な値を**計算**する。 | `calculateDistance()` (距離を計算する) |
| **`validate`** | 入力値や状態の**妥当性**を検証する。 | `validateConfiguration()` (設定値が正しいか検証) |

## 📝 設計書レベルでの命名のポイント

| ポイント | 詳細 |
| :--- | :--- |
| **動詞の利用** | メソッド名は必ず**動詞**で始める（英語圏の命名規則）。 |
| **具体的な対象** | 抽象的な名前ではなく、`isDetected()` のように**何を検知するのか**を明確にすると良いでしょう。例: `isWaterDetected()`、`isMotionDetected()`。 |
| **Camel Case** | 一般的に、JavaやC#、JavaScriptなどでは `isSensorActive` のような**ローワーキャメルケース**（最初の単語の頭文字のみ小文字）が使われます。 |

どのパターンを使用するかは、プロジェクトや使用言語の慣習に従うのが最も良い方法です。

## 💡 状況別として：JUDGEの代わりに使うべき言葉

「JUDGE（判断・判定）」が意図する処理に応じて、以下のより具体的な言葉に置き換えて設計すべきです。

| 意図する処理 (JUDGE) | 推奨されるメソッド名/プレフィックス | 命名例 | 責務の明確化 |
| :--- | :--- | :--- | :--- |
| **真偽の判定** | `is` または `can` | `isDetected()`<br>`canProceed()` | 単に条件を満たすかどうかを $\text{boolean}$ で返す。 |
| **妥当性の検証** | `validate` または `check` | `validateInput()`<br>`checkRange()` | 入力値や状態がルールを満たすか確認し、**無効ならエラーを返す**。 |
| **評価/採点** | `score` または `evaluate` | `calculateScore()`<br>`evaluateRisk()` | 定量的な**評価値や点数**を計算して返す。 |
| **分類/決定** | `determine` または `resolve` | `determineCategory()`<br>`resolveStatus()` | 複数の選択肢の中から**一つに絞って決定**し、その結果（文字列、Enumなど）を返す。 |
